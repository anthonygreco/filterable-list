## Install
```sh
npm install
```

## Development
```sh
npm start
```

## Build
```sh
npm run build
```

## Note on Semantic UI CSS
*Any components used need to exist within the components array in `semantic.json` and the Semantic UI build run.*
```sh
cd src/semantic
gulp build
```
